# FieldTrip
- 골프 조인 SNS 애플리케이션입니다. 
- 사용자는 골프 라운딩을 개최하거나 원하는 라운딩에 조인할 수 있으며, 골프와 관련된 포스트를 작성하거나 구경하는 등 다른 사용자들과 소통할 수 있습니다.
